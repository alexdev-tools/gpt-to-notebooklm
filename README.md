# GPT → NotebookLM Exporter — Пошаговая инструкция

## Что делает расширение

Расширение парсит диалог из ChatGPT и сохраняет его в файл (Markdown, TXT или JSON), который можно загрузить в Google NotebookLM как источник для анализа.

---

## Часть 1. Установка расширения

### Шаг 1. Скачайте и распакуйте архив

Скачайте файл `gpt-to-notebooklm.zip` и распакуйте его в удобное место на компьютере. Внутри будет папка `gpt-to-notebook` со следующей структурой:

```
gpt-to-notebook/
├── manifest.json       ← конфигурация расширения
├── popup.html          ← интерфейс панели управления
├── css/
│   └── popup.css       ← стили интерфейса
├── js/
│   ├── background.js   ← фоновый скрипт (скачивание файлов)
│   ├── content.js      ← парсер страницы ChatGPT
│   └── popup.js        ← логика панели управления
└── icons/
    ├── icon48.png
    └── icon128.png
```

> **Важно:** не переименовывайте и не перемещайте файлы внутри папки. Chrome загружает расширение по структуре `manifest.json`.

### Шаг 2. Откройте страницу расширений Chrome

Введите в адресной строке Chrome:

```
chrome://extensions/
```

и нажмите Enter. Откроется страница управления расширениями.

### Шаг 3. Включите режим разработчика

В правом верхнем углу страницы найдите переключатель **«Режим разработчика»** (Developer mode) и включите его. Появятся дополнительные кнопки вверху страницы.

### Шаг 4. Загрузите расширение

Нажмите кнопку **«Загрузить распакованное расширение»** (Load unpacked). В открывшемся диалоге выберите папку `gpt-to-notebook` (именно папку, не отдельные файлы внутри неё).

### Шаг 5. Проверьте установку

После загрузки расширение появится в списке с названием **«GPT → NotebookLM Exporter»**. Убедитесь, что переключатель активации включён (синий).

В панели расширений Chrome (иконка пазла справа от адресной строки) найдите расширение и нажмите значок **📌 (Закрепить)**, чтобы иконка была всегда видна.

## Часть 2. Использование

### Шаг 1. Откройте диалог в ChatGPT

Перейдите на [chatgpt.com](https://chatgpt.com) и откройте нужный диалог. Дождитесь полной загрузки страницы — все сообщения должны быть видны на экране.

> **Совет:** если диалог длинный, прокрутите страницу до конца, чтобы все сообщения загрузились в DOM. Расширение парсит только то, что отображено на странице.

### Шаг 2. Сканируйте диалог

Нажмите на иконку расширения **G→N** в панели Chrome. В открывшемся окне нажмите кнопку **🔍 Сканировать**.

Расширение проанализирует страницу и покажет:

- Название диалога
- Количество найденных сообщений
- Превью первых сообщений (начало каждого)
- Метод парсинга (для диагностики)

### Шаг 3. Выберите формат и экспортируйте

Нажмите одну из кнопок экспорта:

| Формат | Расширение файла | Когда использовать |
|--------|-----------------|-------------------|
| **Markdown** | `.md` | Лучший вариант для NotebookLM. Сохраняет структуру с заголовками и разделителями |
| **Текст** | `.txt` | Универсальный формат, если Markdown не подходит |
| **JSON** | `.json` | Для программной обработки или интеграции с другими инструментами |

Chrome предложит сохранить файл — выберите место и нажмите **Сохранить**.

### Шаг 4. Загрузите файл в NotebookLM

1. Откройте [notebooklm.google.com](https://notebooklm.google.com)
2. Создайте новый блокнот или откройте существующий
3. Нажмите **«+ Добавить источник»** (Add source)
4. Выберите **«Загрузить файл»** (Upload file)
5. Выберите скачанный файл
6. NotebookLM проиндексирует диалог как источник

## Часть 3. Устранение проблем

### «Сообщения не найдены»

Расширение использует 4 стратегии парсинга DOM ChatGPT. Если ни одна не нашла сообщений:

- Убедитесь, что вкладка с ChatGPT активна и диалог загружен
- Попробуйте перезагрузить страницу ChatGPT (F5) и повторить сканирование
- Прокрутите диалог до конца, чтобы все сообщения отрисовались

### «Ошибка: Could not establish connection»

Расширение не может связаться со скриптом на странице:

- Перезагрузите страницу ChatGPT
- Если не помогло — зайдите в `chrome://extensions/`, отключите и снова включите расширение

### Мало сообщений в экспорте

ChatGPT использует виртуальный скроллинг — не все сообщения присутствуют в DOM одновременно. Прокрутите диалог сверху вниз перед сканированием.

### Кракозябры в файле

Все файлы экспортируются в кодировке UTF-8. Если при открытии видите кракозябры, укажите кодировку UTF-8 в текстовом редакторе.

## Часть 4. Структура экспортированного файла

### Markdown (.md)

```
# Название диалога

> Экспорт из ChatGPT | 24.03.2026, 12:30:00 | 15 сообщений
> URL: https://chatgpt.com/c/...

---

## 👤 Пользователь

Текст вопроса пользователя...

---

## 🤖 ChatGPT

Текст ответа ChatGPT...

---
```

### JSON (.json)

```json
{
  "title": "Название диалога",
  "url": "https://chatgpt.com/c/...",
  "exported_at": "2026-03-24T12:30:00.000Z",
  "message_count": 15,
  "messages": [
    { "role": "user", "content": "Текст вопроса..." },
    { "role": "assistant", "content": "Текст ответа..." }
  ]
}
```

## Часть 5. Обновление расширения

При получении новой версии:

1. Распакуйте новый архив, заменив старую папку `gpt-to-notebook`
2. Откройте `chrome://extensions/`
3. Найдите расширение и нажмите **🔄 (Обновить)** или кнопку перезагрузки
4. Перезагрузите открытые вкладки ChatGPT

---

## Безопасность и приватность

- Расширение работает **полностью локально** — данные не отправляются на внешние серверы
- Парсинг происходит в браузере, файл сохраняется на ваш компьютер
- Расширение имеет доступ только к страницам `chatgpt.com` и `chat.openai.com`
- Исходный код открыт — можно проверить содержимое всех файлов перед установкой
